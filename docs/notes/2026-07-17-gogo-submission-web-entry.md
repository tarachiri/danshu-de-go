# 2026-07-17 GOGO投稿受付Web入口

## 課題

断酒でGO!!本体サイトから、例会予定表や直接入力をGOGOレビューへ送る入口がなかった。

## 変更内容

- `gogo-submit.html`を追加した。
- PDF、Excel、Word、画像の複数添付、直接テキスト、補足コメントに対応した。
- 高齢利用者を意識し、大きめの文字と単一の送信ボタンで構成した。
- `js/menu.js`のメニューに「例会情報を送る」導線を追加した。
- 投稿は`https://gogo-review.nukadokonokai.com/submit`へ直接送信し、公開JSONへは自動反映しない。

## 確認

- API側でファイルまたは直接テキストのどちらかを必須にする。
- 投稿成功時は受付番号を表示し、失敗時は同じ画面に理由を表示する。

## 次の課題

- 公開後、スマートフォン実機でファイル複数選択と写真投稿の操作性を確認する。
