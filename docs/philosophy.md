# 断酒でGO!! 開発哲学 / Philosophy of Dansyu-de-GO!!

> 「日本全国の断酒会を一つに！心の連鎖握手」

---

きっかけは「全部」だった。

自分や家族がアルコール問題を経験したこと。
断酒会の活動に関わったこと。

「全員」に届けたい。

今まさに飲んでいる本人。
そばで見守る家族・友人・職場の人。
回復の途中で孤立している人。

立場上、それ以外を積極的に案内することはできない。

どちらが合うかは人それぞれでいい。

**どこでもいいから、すぐに繋がってほしい。**

地図を開いて、一番近い場所に、今週中に行ってみてほしい。
それだけでいい。

## 断酒でGO!!とは何か

全国の断酒会例会場を地図で見えるようにするPWA（Progressive Web App）だ。

「近くに断酒会があること」を知らずに孤立している人がいる。
公式サイトがない県、情報が古い県、検索しても出てこない地域がある。

日本の断酒会には三つのエリアがある。

- **自立エリア**：公式サイトで情報発信できている
- **依存エリア**：ソーバーねっと（addiction-peer.net）に情報を預けている
- **瀕死エリア**：どこにも情報が存在しない、消えかかっている

断酒でGO!!は、この三つすべてに届く地図になりたい。

自立エリアには「もっと見つけてもらえる場所」として。
依存エリアには「より使いやすい窓口」として。
瀕死エリアには「最後の灯台」として。

---

## データ品質への誓い

間違った場所への案内を、最も警戒している。

アルコール依存症から回復しようとしている人が、
勇気を振り絞って例会に向かって、
そこに誰もいなかったら。

それは情報インフラの失敗ではなく、人への裏切りだ。

だから：

- 公式サイトの情報を「真実」とする
- 確認できないデータは `needs_verification` フラグで明示する
- 幽霊データより、少ないデータの方がまし
