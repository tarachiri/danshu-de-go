🦆 断酒でGO!! 開発ノート

2026年6月28日｜かもちゃん記

全断連加盟断酒会 公式サイト調査完了

断酒でGO!!の新着タブ強化に向けて、公益社団法人全日本断酒連盟（全断連）に加盟する各都道府県断酒会の公式ウェブサイトを全件調査した。アルコール依存症からの回復を支える断酒会の情報インフラとして、断酒でGO!!がより多くの情報を集約できる体制を整えることが目的だ。

断酒例会への参加は、アルコール依存症からの回復において非常に重要な役割を果たす。しかし全国各地で開催される例会やセミナー・講演会などの情報は各団体のウェブサイトに散在しており、一元的に把握することが難しい。断酒でGO!!はこの課題を解決するための「人命インフラ」として機能する。

DBスキーマ

調査結果はdanshu.dbのdanshu_sitesテーブルに格納した。 

CREATE TABLE danshu_sites (
  id            INTEGER PRIMARY KEY AUTOINCREMENT,
  pref          TEXT,         -- 都道府県
  org_name      TEXT,         -- 団体名
  org_name_full TEXT,         -- 正式名称（法人格含む）
  url           TEXT,         -- 公式サイトURL
  news_url      TEXT,         -- 新着ページURL
  rss_url       TEXT,         -- RSSフィードURL
  is_active     INTEGER DEFAULT 1,
  last_checked  DATE,
  note          TEXT,
  created_at    TEXT,
  updated_at    TEXT
);

注意事項

⚠️ 京都断酒平安会（id=10）
最近の投稿にスパムと思われる英数字タイトルが混入している。RSS取得時にタイトルフィルタリングが必要。
⚠️ 文字化けサイト（id=4・9・24）
東京多摩断酒連合・滋賀県断酒同友会・高知県断酒新生会はShift-JISエンコードのため、スクレイピング時にencoding='shift-jis'対応が必要。

次のステップ

断酒からの回復を支える情報インフラとして、断酒でGO!!はアルコール依存症に苦しむ本人・家族が必要な情報に素早くアクセスできる環境を目指している。次フェーズでは以下を実装する。

  1. RSS取得スクリプト（13件・自動化容易）
  2. スクレイピングスクリプト（15件・サイト別対応）
  3. generate_news.pyへの組み込み
  4. news.jsonへの反映・新着タブでの表示

全国の断酒会例会情報・セミナー・講演会・研修会などをリアルタイムで集約することで、アルコール依存症からの回復を求める人々が「今日開催される例会」をすぐに見つけられるアプリを実現する。断酒継続のサポートとして、断酒でGO!!は日本全国の断酒会を一つにつなぐ。
